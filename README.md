# AmplyAI Chatbot (DIY)

A Next.js chatbot app that connects directly to OpenAI’s API.

## Features
- Next.js frontend + API route
- Calls OpenAI via `/api/chat`
- Resume helper demo (job role + tone + resume text → improved resume)
